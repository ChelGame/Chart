# 📊 Аналитика 4 параметров — интерактивный график

Это React-приложение для визуализации данных (Cost, CPA, ROI, Conversions) с кастомными маркерами и динамической толщиной линии. Проект собран на **Create React App** и использует **TypeScript**, **Tailwind CSS**, **Recharts**, **react-hook-form** и **zod**.

---

## 🚀 Быстрый старт (для разработчиков с Node.js)

1. **Установите Node.js** (LTS-версия) с [официального сайта](https://nodejs.org/).
2. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone <your-repo-url>
   cd project-folder
Установите зависимости:

bash
npm install
Запустите локальный сервер разработки:

bash
npm start
Откройте http://localhost:3000 в браузере.
